⚙️ One-time setup needed

This site needs to know where to fetch the scores CSV from. Follow these steps once — after that, just upload a new CSV to update everyone's view.

1
Go to github.com and create a free account (or sign in if you have one).
2
Create a new public repository — name it something like on-the-ball.
3
Upload this file (rename it index.html) and your scores CSV (name it scores.csv) into the repository.
4
In the repository, go to Settings → Pages, set Source to Deploy from a branch, branch = main, folder = / (root), and click Save.
5
Open index.html in GitHub, click the pencil (edit) icon, and change the line near the top that says:

DATA_URL: null

to:

DATA_URL: 'https://raw.githubusercontent.com/YOUR_USERNAME/on-the-ball/main/scores.csv'

Replace YOUR_USERNAME with your GitHub username. Commit the change.
6
Your site will be live at https://YOUR_USERNAME.github.io/on-the-ball — share this link with your group!

To update scores: Export a new CSV from Excel, go to your GitHub repo, click on scores.csv, then click the upload/replace icon to drop the new file in. The site updates within a minute.
